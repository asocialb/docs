### Выбор сети для CSS

#### Варианты сети

| Тип сети | Доступность | Стабильность |
| --- | --- | --- |
| Проводная сеть | Низкая | Высокая |
| Wi-Fi | Высокая | Низкая |
| По возможности рекомендуется использовать проводную сеть, которая более стабильна, чем Wi-Fi, и менее подвержена помехам. Wi-Fi больше подходит для прямых трансляций мероприятий благодаря удобству использования. |  |  |

#### Тест пропускной способности исходящего канала

Требования к пропускной способности исходящего канала зависят от качества видео и разрешения. Как правило, более высокое качество видео с более высоким разрешением означает более высокие требования к пропускной способности исходящего канала. Пропускная способность исходящего канала не должна быть менее 1 Мбит/с. Для проверки состояния пропускной способности исходящего канала можно воспользоваться

speedtest

.

![](https://mc.qcloudimg.com/static/img/b5724af9873220c395e295894205e4ad/image.png)

### Установка программного обеспечения для push

#### Установка OBS

Скачайте установочный пакет OBS на

официальном сайте OBS

и установите его с настройками по умолчанию. Поддерживаются операционные системы Windows, Mac и Linux. Убедитесь, что загруженное программное обеспечение — Open Broadcaster Software. OBS Studio также доступен, но не рассматривается в данном документе.

![](https://mc.qcloudimg.com/static/img/dcbb929e364b1d8e80c04e326a756a26/image.png)

#### Установка XSplit

При необходимости скачайте установочный пакет XSplit на официальном сайте XSplit и установите его с настройками по умолчанию. XSplit — платное программное обеспечение. При ограниченном бюджете рекомендуется использовать OBS (

**бесплатно**

). Для игровых CSS-трансляций доступен отдельный установочный пакет XSplit. Для неигровых CSS-трансляций рекомендуется BroadCaster.

![](https://mc.qcloudimg.com/static/img/18c47cb7646e189acc168e6a5e8e4714/image.png)
