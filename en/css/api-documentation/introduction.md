# Introduction

Leveraging Tencent’s powerful technologies and the capabilities of its core audio/video services, CSS (Cloud Streaming Services) offers professional, stable, and fast live streaming and content distribution services that guarantee ultra-low latency and are capable of handling a large number of concurrent requests.

Before using the APIs of CSS, make sure you have read and agree to its [billing rules](https://intl.cloud.tencent.com/document/product/267/2819).

> **Notes:**
> 
> All APIs in this section are version 3.0. New CSS APIs added in the future will also be version 3.0. We strongly recommend you use 3.0 APIs.
> Legacy APIs remain available but will not be updated. For more information about legacy APIs, see CSS APIs (Legacy) Overview.
