# Live Quality Control Configuration

The Quality Control and Assessment feature for live streaming push is disabled by default. This document will guide you on how to enable the Quality Control and Assessment feature for a specified push domain by associating it with a Quality Control and Assessment template, as well as how to unbind the template to disable the feature after successful association.

## Notes

- A template takes effect about 5-10 minutes after it is bound to a domain.
- After successfully associating the template, enable the Quality Control and Assessment feature for the streaming address under the specified push domain.
- A domain name can be associated with only one Quality Control and Assessment template. Once associated, all streams under that domain will adhere to the Quality Control and Assessment tasks defined by the template.

## Prerequisites

- You have logged in to the [CSS console](https://console.tencentcloud.com/live) and added a [push domain](https://intl.cloud.tencent.com/document/product/267/35970).
- You have created a [Quality Control and Assessment template](https://www.tencentcloud.com/document/product/267/73200#.E5.88.9B.E5.BB.BA.E6.97.B6.E7.A7.BB.E6.A8.A1.E6.9D.BF).

## Binding a  Quality Control and Assessment Template

1. Go to [Domain Management](https://console.tencentcloud.com/live/domainmanage), and click the target **push domain** or click **Manage** to enter the domain details page.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/300751aa79af11f0914f52540099c741.png)

2. Select the **Template Configuration** tab and click **Edit**in the **Live Quality Control Configuration** area.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7f134ffcab0711f09710525400e889b2.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a0b0508fab0711f0a68e5254001c06ec.png)

3. Select a Quality Control and Assessment template and click **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3200f331ab0811f0b08552540044a08e.png)

## Unbinding a Quality Control and Assessment Template

1. Go to [Domain Management](https://console.tencentcloud.com/live/domainmanage), and click the target **push domain** or click **Manage** to enter the domain details page.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3693cbc479af11f08255525400e889b2.png)

2. Select the **Template Configuration** tab and click **Edit** in the **Live Quality Control Configuration** area.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5c6ad4e6ab0811f0b5345254005ef0f7.png)

3. Unselect the template and click **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6e01b169ab0811f096c2525400454e06.png)
