# Use Cases

## Live Game Streaming

LVB provides upstream acceleration, high-quality transcoding, and accelerated distribution services for live game streaming platformsr. TSC transcoding technology can reduce the bit rate by up to 50%+ under the same image quality, helping to reduce bandwidth costs. In addition, it also has the capabilities of multi-image cloud mixing, voice commentary to real-time subtitles, time shifting, and  highlight editing to help achieve a rich game live broadcast experience.

You can integrate LVB with MLVB SDK to quickly realize smooth live game streaming with low lag and low stutter.

## Live Show

LVB provides features such as quick live streaming startup, color filters, and manual exposure for live show. It is easy to use and ready to go, enhancing the popularity and atmosphere of live broadcasts. At the same time, it supports content security audits, allowing for the detection of images, audio, and text, thereby reducing the platform's auditing costs.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/33ea58b5911e11ef8a1b525400bdab9d.jpeg)

## Online Education

LVB provides a one-stop service for online education. Your well-crafted teaching materials can be uploaded and downloaded through the agile console and APIs, and are protected with player passwords and dynamic URL-based hotlink protection. It also supports cloud transcoding, recording, time-shifting and other functions, allowing students to review course content at different bit rates and resolutions at any time and consolidate their learning results.

In addition, the open player APIs make it easy to implement various teaching features such as setting playback time limits and disallowing progress bar dragging.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/33ee8f1e911e11ef8a1b525400bdab9d.jpeg)

## Radio, TV, and New Media

LVB provides a complete set of live video streaming services for the radio, it ensures stable transmission in weak network environments and supports high-definition resolutions such as 2K and 4K. Combined with capabilities such as code rate adaptation, Live Subtitling, Standby Streams, and director stations, it can quickly create professional live broadcasts suitable for news, media, and variety shows. Service platform to realize the low-threshold transformation from traditional media to Internet media.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/33ee3d7b911e11efafd5525400d5f8ef.jpeg)

## Live Shopping

To ensure real-time interaction between anchors and viewers, and combined with Tencent Cloud IM, LVB provides a one-stop selling + chat interaction solution, allowing merchants to more comprehensively deliver product information, promote effective decision-making by users, reduce marketing costs, and increase transaction volume.
