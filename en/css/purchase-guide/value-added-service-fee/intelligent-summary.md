# Intelligent Summary

Using large model capabilities, this feature generates paragraph topics and summary information from videos, making it suitable for various scenarios such as meetings and online education. Intelligent Summary is a paid value-added service. Below are the billing items for the value-added features generated by using the Intelligent Summary function.This document describes how the feature is billed.

## Must-Knows

In addition to value-added service fees, the use of standby streams may also incur [traffic/bandwidth costs](https://www.tencentcloud.com/document/product/267/2819).

## Pricing

| Type | Unit cost |
| --- | --- |
| Price (USD/Billing Unit) | 0.01515 |

**Ratio of billing duration to actual duration:**

| Value-Added Feature | RatioBilling Duration (Billing Unit) : Actual Duration (Minutes) |
| --- | --- |
| Intelligent Summary | 2.8 : 1 |

### Billing Overview

- Billing item: Basic fee for value-added features.
- Billing mode: Postpaid.
- Billing rules: Billed by streaming duration.
- Billing cycle: Daily. The value-added service fee for each day will be deducted when the bill is issued on the next day. The billing details and bill issuance time shall be subject to the actual bill.

### Calculation Formula

- Billing usage = Actual usage × Coefficient
- Value-added service fee = Basic fee for the value-added service × Billing usage

### Billing Example

If you streamed for 60 minutes on February 5, 2026—meaning you used the Intelligent Summary feature for 60 minutes—the bill generated on February 6, 2026, would be:

0.01515 (USD/Billing Unit) × 2.8 (Billing Unit/minute) × 60 (minutes) = 2.5452(USD)
