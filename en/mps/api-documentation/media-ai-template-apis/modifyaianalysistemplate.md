# ModifyAIAnalysisTemplate

## 1. API Description

Domain name for API request: mps.intl.tencentcloudapi.com.

This API is used to modify a custom content analysis template.

Note: templates with an ID below 10000 are preset and cannot be modified.

A maximum of 10 requests can be initiated per second for this API.

## 2. Input Parameters

| Parameter Name | Required | Type | Description |

## 4. Example

### Example1 Disabling a Smart Cover Task

This example shows you how to modify a custom video content analysis template to disable the smart cover task.

#### Input Example

```
https://mps.intl.tencentcloudapi.com/?Action=ModifyAIAnalysisTemplate
&Definition=30
&CoverConfigure.Switch=OFF
&<Common request parameters>
```

#### Output Example

```json
{
    "Response": {
        "RequestId": "12ae8d8e-dce3-4151-9d4b-5594145287e1"
    }
}
```

### Example2 Enabling and Disabling a Content Analysis Task Simultaneously

This example shows you how to modify a custom video content analysis template to enable the smart tag task and disable the smart cover task.

#### Input Example

```
https://mps.intl.tencentcloudapi.com/?Action=ModifyAIAnalysisTemplate
&Definition=30
&TagConfigure.Switch=ON
&CoverConfigure.Switch=OFF
&<Common request parameters>
```

#### Output Example

```json
{
    "Response": {
        "RequestId": "12ae8d8e-dce3-4151-9d4b-5594145287e1"
    }
}
```

### Example3 Enabling a Smart Cover Task

This example shows you how to modify a custom video content analysis template to enable the smart cover task.

#### Input Example

```
https://mps.intl.tencentcloudapi.com/?Action=ModifyAIAnalysisTemplate
&Definition=30
&CoverConfigure.Switch=ON
&<Common request parameters>
```
