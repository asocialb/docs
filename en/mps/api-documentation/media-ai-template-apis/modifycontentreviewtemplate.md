# ModifyContentReviewTemplate

## 1. API Description

Domain name for API request: mps.intl.tencentcloudapi.com.

This API is used to modify a custom content moderation template.

A maximum of 10 requests can be initiated per second for this API.

We recommend you to use API Explorer

Try it

API Explorer provides a range of capabilities, including online call, signature authentication, SDK code generation, and API quick search.  It enables you to view the request, response, and auto-generated examples.

## 4. Example

### Example1 Modifying a Content Moderation Template to Enable Porn Detection

This example shows you how to modify a custom AI content moderation template to enable porn detection. The default thresholds for rule violation and manual recognition are used, and no tags are specified.

#### Input Example

```
https://mps.intl.tencentcloudapi.com/?Action=ModifyContentReviewTemplate
&Definition=30
&PornConfigure.ImgReviewInfo.Switch=ON
&<Common request parameters>
```

#### Output Example

```json
{
    "Response": {
        "RequestId": "67ae8d8e-dce3-4151-9d4b-5594145287e1"
    }
}
```

### Example2 Modifying a Content Moderation Template to Enable Porn Detection with Specified Thresholds and Tags

This example shows you how to modify a custom AI content moderation template to enable porn detection. The thresholds for rule violation and manual recognition are set to 90 and 60, respectively, and the tag is specified as `sexy`.

#### Input Example

```
https://mps.intl.tencentcloudapi.com/?Action=ModifyContentReviewTemplate
&Definition=30
&PornConfigure.ImgReviewInfo.Switch=ON
&PornConfigure.ImgReviewInfo.LabelSet.0=sexy
&PornConfigure.ImgReviewInfo.BlockConfidence=90
&PornConfigure.ImgReviewInfo.ReviewConfidence=60
&<Common request parameters>
```

#### Output Example

```json
{
    "Response": {
        "RequestId": "82ae8d8e-dce3-4151-9d4b-5594145223e1"
    }
}
```
