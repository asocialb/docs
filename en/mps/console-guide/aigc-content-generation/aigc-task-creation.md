# AIGC Task Creation

## **Scenarios**

The Media Processing Service (MPS) console offers different types of AIGC generation task experiences. Users can quickly experience the generation effects through the console.

## Prerequisites

1. You have [registered a Tencent Cloud](https://www.tencentcloud.com/document/product/378/17985) account and completed identity verification.
2. You have activated [Tencent Cloud MPS](https://www.tencentcloud.com/document/product/1041/44041).

## Usage Steps

1. Log in to the **MPS console**, go to the [**AIGC Task Creation**](https://console.tencentcloud.com/mps/aigc/create) page, and select **Video Generation** or **Image Generation**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2919adaee16511f0ab0f5254005ef0f7.png)

3. You can customize the resolution and ratio of the generated video. During video generation, the duration can be selected.

> **Note:**The parameter value ranges supported by different models vary.

4. Click **Generate**, and then you can stay on the current page to generate other content or go to the AIGC task management page to view the task.
