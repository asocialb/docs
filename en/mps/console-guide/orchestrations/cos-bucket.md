# COS Bucket

## Overview

The Cloud Object Storage (COS) Bucket provides a list of input and output buckets bound with Media Processing Service (MPS) schemes, making it easier to query the usage of related buckets.

## Details

1. Log in to the [**MPS console**](https://console.tencentcloud.com/mps), and click [Orchestrations > COS Bucket](https://console.tencentcloud.com/mps/workflows/buckets) to enter the COS Bucket page. On this page, you can click  **Source buckets**  and  **Output bucket**  to view the input and output buckets bound with MPS.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/558c9d93286411ef812f5254002a8f58.png)

2. Click **View files** in the operation column to view the files in a bucket.
![](https://staticintl.cloudcachetci.com/cms/backend-cms/fec701b1d6d811eeb0d9525400461a83.png)

> **Note:**This page shows only buckets bound with MPS schemes. To create buckets or perform other bucket operations, please go to the [COS console](https://console.tencentcloud.com/cos/bucket).
