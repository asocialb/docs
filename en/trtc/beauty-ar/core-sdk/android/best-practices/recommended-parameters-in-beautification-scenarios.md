# Recommended Parameters in Beautification Scenarios

The one-click beauty effect parameter table in the demo project is as follows. If you want to implement the one-click beauty effect in your application, use the following parameters to configure the beauty effect.

| Feature | iOS Parameter Recommendation | Android Parameter Recommend |
| --- | --- | --- |
| Brighten/Bright White | 40 | 40 |
| Smooth Skin/Young | 40 | 40 |
| Sharpness | 80 | 80 |
| Sharpen | 30 | 30 |
| Face Width | 5 | 5 |
| Thin Face/Natural | 30 | 30 |
| V Shape | 20 | 20 |
| Smile Lines | 30 | 30 |
| Big Eyes | 20 | 20 |
| Bright Eyes | 40 | 40 |
| Eye Bags | 50 | 50 |
| Nose Size | 20 | 20 |
| White Teeth | 40 | 40 |
