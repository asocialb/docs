# Official License

Beauty AR SDK supports integration with both **Web & H5** and **Mobile**platforms. This article will guide you through **Web&H5 platform** [Purchasing an official license](#webpurchaseofficial), [Update official license valid period](#webupgradeofficial) and [Auto-renewal](#webrenewal).

## Web&H5

### Purchase an official license

You can choose and purchase the SDK package that suits your needs from the [Beauty AR SDK Purchase Page](https://console.trtc.io/subscription/buy/beauty) and obtain the official license for use. The **validity period starts from the time of purchase, so please use it as soon as possible**. For feature differences between different SDK versions, please refer to [Billing overview](https://www.tencentcloud.com/document/product/1143/60214#).

The detailed directions are as follows:

1. Go to [TRTC Console > Relevant Services](https://console.trtc.io/aboutcloud), click **Get Started** to enter License Management, then click **Create official license**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cef683f4053311efa7e752540052a3fc.png)

2. Select the bundle package version, enter the official app's `Project Name `and `Domain` click **Bind** to choose an **unbound** Beauty AR SDK Web&H5 package (if no bindable packages are available, you can go to the [Beauty AR SDK Purchase Page](https://console.trtc.io/subscription/buy/beauty) to buy one), then click **Confirm**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/820292db0c4c11ef9ef35254002977b6.png)

3. After the official license is successfully created, the page will display the information of the generated official license. You can refer to the **Integration Guide** below for integration. The `License Key` and `License Token` will be used in the integration guide.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a3f2b87e054411ef8a9c5254003e1eb1.png)

### Update official license valid period

You can log in to  [TRTC Console > Relevant Services](https://console.trtc.io/aboutcloud), click **Get Started** to enter the **License Management** page to check the validity period of the Beauty AR SDK Web&H5 official license. Official Beauty AR Licenses will send you an expiry reminder 32 days, 7 days, 3 days, and 1 day before expiration, reminding you to renew on time to avoid affecting normal business operations. If your Beauty AR SDK Web&H5 official license is about to expire, you can follow these steps to renew:

1. Select the license you wish to renew, and click **Update Validity** at the bottom of the package information box.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5a4132f2054511ef99495254007c64fd.png)

2. Beauty AR License supports two ways to renew the validity period: **Renew the current license** and **Select another license resource to replace**, as follows.

> **Note:****Resources with auto-renewal enabled do not support the license resource replacement method for renewal.** If you want to change the validity period to that of another license, disable the auto-renewal function.![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9635cea9060711ef9aef5254009d370c.png)

Renew the current license

Select another license resource to replace

1. Select **Renew the current license**, then click **Confirm**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6e9896ef053b11ef9f9c525400d4e181.png)

> **Note:**If selecting **Renew the current license**:when the license resource is within the valid period or has expired for less than seven days, according to Tencent Cloud's billing regulations, renewing for the current license is supported. After renewal, the valid period will be refreshed to: current validity period + renewal duration.When the license resource has expired for more than seven days, the original resource has been destroyed, and we will purchase the same type of resource for you and automatically complete the replacement binding. After renewal, the valid period will be refreshed to: payment date + renewal duration.

2. On the **Renewal** page, select the **Renewal period**. The Beauty AR Web endpoint renews periodically **By Month**. Click **Confirm** to extend the license validity.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dae0a007054411efa7e752540052a3fc.png)

1. Select **Select another license resource to replace**, then click **Confirm**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/e163b94e053b11ef81955254003a4131.png)

2. On the **Update validity** page, click **Bind**, select the **Unbound** Beauty AR SDK Web&H5 Package (if there are no bindable packages available, go to the [Beauty AR SDK Purchase Page](https://console.trtc.io/subscription/buy/beauty) to buy one), then click **Confirm**.

> **Note:**Currently, extending the validity of the package only supports the same type of package. That is, if the bound license package type is Standard, then the renewal can only select a Standard package for renewal. If you want to change the bound package type, you need to [Contact us](https://trtc.io/contact) or contact the business team for processing.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/31b0f95b05fa11ef8b3f5254007c64fd.png)

3. Check the validity period after the update.

> **Note:****Beauty AR Official License does not support information modification**. If you need to modify License information, please do not use the purchased resource pack for updating the License validity period. Instead, click **Create official license** to recreate the application, add a new License, and bind new package name information.

### Auto-renewal

It is advised to use the **Enable Auto-Renewal** option for renewal. Once enabled, it will automatically renew monthly after expiration, provided your account has a **Sufficient balance**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6b59fb03054411efa7e752540052a3fc.png)
