# Package Management

TRTC supports viewing the usage of Package under your account through the console.  You can click [Package Management > My Package](https://console.trtc.io/package) in the left sidebar to view package info in the Package List.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/526ba4b6c06211f0b4c35254001c06ec.png)

## Package list

> **Noteï¼**Package types are divided into Free Trial and monthly package, the former can deduct audio and video, recording, mixtranscoding time, the latter can deduct audio and video time, the specific deduction ratio details can be seen [free time description](https://trtc.io/document/42735).

| Features | Description |
| --- | --- |
| Package Name | The purchased package name is supported as follows:RTC Engine - StandardRTC Engine - ProTRTC Free Trial1-to-1 CallGroup Call |
| Package ID | The system automatically generates a unique package ID after a successful purchase |
| Package Type | There are two types of free time package and monthly package, among which:Standard/Professional packages are customer purchased products, including the Call/RTC EngineThe Free Trial is a platform giveaway product, distributed for 12 months at a time, 10,000 minutes per month |
| Package Duration(min) | Some minutes are included in the Package |
| Remainging time of this cycle(minutes)/(%) | With the number of minutes remaining and the proportion of usage, you can see the usage of the package more intuitively |
| Start time of this cycle | The time the package was successfully purchased |
| End time of this cycle | Package expiration time |
| Status of this cycle | Refunded, Expired, Run out, Not in Use, In Use |

## Purchase package

If your actual business needs require renewing a package, go to [Package Management > Package Subscription](https://console.trtc.io/subscription), select **the product for which you need to purchase a package**, choose **the version you need to purchase**, and then click **Subscribe Now** to complete the purchase.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7474f464cb6611f0b011525400bf7822.png)

> **Noteï¼**Before purchasing a package, make sure you understand the billing information. For details, see [Billing Overview](https://trtc.io/document/34610).
