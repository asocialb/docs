# On-Cloud MixTranscoding

On-Cloud MixTranscoding consumption statistics of TRTC console are mainly used to show the consumption details generated by [MCU cloud mixtranscoding](https://trtc.io/document/47631) provided by TRTC. If you use the mixtranscoding function billing provided by CSS, please go to the [CSS console](https://console.tencentcloud.com/live/livestat) to check.

## Notes

The data is updated every 5 minutes instead of in real-time, so there may be a delay of 5-180 minutes in the display of statistics.

## Directions

1. Enter the [Tencent RTC console > Usage](https://console.trtc.io/statistics) and select **Cloud-Based Mixed-Stream Transcoding**.
2. Select the application you want to view, then select the time period.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/4d8f2b25cb6811f0a93d52540044a08e.png)

## Usage Details

Durations are calculated in seconds and rounded up to minutes. Therefore, the cumulative duration used for billing may differ slightly from your actual usage. For the actual billable amount, see the bills generated in [Billing Center](https://console.intl.cloud.tencent.com/expense/bill).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cd374169cf2611f0800752540099c741.png)
