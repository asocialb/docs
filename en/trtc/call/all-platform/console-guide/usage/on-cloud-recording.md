# On-Cloud Recording

On-Cloud Recording usage statistics of the TRTC console are used to display details on the cloud recording usage of TRTC by [recording duration](https://trtc.io/document/52817#.E4.BA.91.E6.9C.8D.E5.8A.A1.E6.B6.88.E8.80.97). If you are using the recording [concurrent peak route](https://www.tencentcloud.com/document/product/267/39605) billing for CSS, you need to go to the [CSS console](https://console.tencentcloud.com/live/livestat) to check.

## Note

The data is updated every 5 minutes instead of in real-time, so there may be a delay of 5-180 minutes in the display of statistics.

### Directions

1. Enter [Tencent RTC console > Usage](https://console.trtc.io/statistics), select **On-Cloud Recording**.
2. Select the application you want to view, then select the time period.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2d200b95cb6811f0a4a55254001c06ec.png)

## Usage Details

Durations are calculated in seconds and rounded up to minutes. Therefore, the cumulative duration used for billing may differ slightly from your actual usage. For the actual billable amount, see the bills generated in [Billing Center](https://console.intl.cloud.tencent.com/expense/bill).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1f7c1f72cb6811f0ae4f52540099c741.png)
