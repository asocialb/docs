# New Console Introduction

This document provides an operation guide for the new Chat console to help you quickly understand the usage method of the console and ensure a smooth transition.

> **Note:**The new Chat console retains the account system and CAM authentication capabilities of the Tencent Cloud official website, allowing you to log in with your original account and manage your Chat resources.The new Chat console supports multiple languages, including English, Chinese, Japanese, and Korean. If you encounter any issues during use, please [contact us](https://trtc.io/contact).

## Introduction to the New Console Interface

### 1. Top Navigation

Includes Account Information, Billing Center,Ticket, Docs, and Demo experience gallery.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/33547b08d4c611f08206525400e889b2.png)

### 2. New Layout of the Overview Page

[The console overview](https://console.trtc.io/) includes common modules such as Application Management and Package Management. The new console can manage both Chat and RTC Engine, Call, Conference, Live, Beauty AR, and other product apps, eliminating the need to repeatedly jump around within multiple consoles for users who use multiple products at the same time.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/389e9438d4c611f0a73852540099c741.png)

> **Note:**The current package management page only displays usage-related packages. If you want to view the prepaid billing information and all postpaid billing information for Chat and Beauty AR, please go to [Expense Center > Billing Center](https://console.tencentcloud.com/expense/deal) to view.

### 3. Application

Chat manages different services and programs through [Applications](https://console.trtc.io/app). You need to create an application and open Chat service (receive any version) to access Chat service.

Under the new console, multiple services can be opened under each application. In addition to Chat, Call, Conference, Live, RTC Engine, In-game Voice Chat, etc. can also be managed and opened.

**3.1 Application List**

You can create new apps here and view the basic information of all apps created, including SDKAppID, Status, Deployment Region, etc.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3e3b2c1dd4c611f09f555254007c27c5.png)

**3.2 Application Management Detail**

Click on the application name in the application list to access the application management details page.

- Overview panel: Basic information about the application.
- Product integration: You can enable multiple services under each application. In addition to Chat, you can also manage and enable Call, Conference, Live, RTC Engine, In-game Voice Chat, etc. The details page can be used to view the version information of the service in use, configure related options, and perform upgrade operations.
- Advanced function: Common advanced service settings for this application.
- Development tool: Provides tools to generate and verify UserSig, and generate publishing addresses for the live stream publishing (RTMP+RTC) feature.
- Application management: View application list, manage application info and status, and purchase as well as manage packages.
- Monitoring dashboard & usage statistics: Query real-time call data information and track feature usage.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebf1d8e0d4c611f09975525400bf7822.png)

**3.3 Chat Service Configuration**

Select [**Chat**](https://console.trtc.io/chat) in the left sidebar, select target application at the **top**. Here you can separately configure the chat service for each application.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2507bb3ed18211f0a73852540099c741.png)

## Chat Application Creation Process and Configuration Page Location Adjustment

### Chat Application Creation Process

1. In [My Applications](https://console.trtc.io/app), click **Create Application**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5b0c5d6ed4c711f0931a5254005ef0f7.png)

2. In the pop-up window, enter application name, select product, version and deployment region, then click create.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/73d501c0d1b611f0a78a52540044a08e.png)

> **Note:**Regarding deployment region options.Chat deploys multiple data centers around the world, all of which support global access, but users between different data centers cannot interoperate. Please carefully select the data center that best suits your business based on the distribution of major users or compliance requirements. Once selected, no subsequent changes will be possible.TRTC's data center is Singapore by default and does not support users to choose their own. The data and privacy terms related to international sites can be found in the [Privacy Policy](https://trtc.io/document/48827?product=privacysla).We support global access and are not limited by deployment regions. We have deployed nearly 3,000 acceleration nodes in various countries around the world. When the terminal is logged in, Chat SDK and TRTC SDK will visit the nearest access point or acceleration point to realize accelerated transmission through the optimal scheduling strategy.

### Configure a Chat Application in the Console

Log in to the [Console > Chat Product Details Page](https://console.trtc.io/chat), select the target application at the **top**. The three management modules of the Legacy Console: **Configuration & Management, Plugin Service, Development Assistance**, are all shown herein.

- **Configuration & Management:** Overview, Account Management, Group Management, Feature Configuration, Webhook Configuration, and Data Statistics.
- **Push Service:** Push and plug-in marketplace.
- **Development Assistance:** Real-time monitoring, development tool, and integration guide.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1300d47dd4d711f08206525400e889b2.png)

## FAQs

### Should I Choose to Use the International Site or Domestic Site

A: If you are a non-Chinese mainland entity and need to settle with Tencent Cloud in USD, it is advisable to use the products and services on the [international site](https://trtc.io).

### Q: Are the Services and Features Different

A: The services and features of the domestic site (product named "IM") and the international site (product named "Chat"), as well as their integration and usage, are consistent. Use with confidence.
