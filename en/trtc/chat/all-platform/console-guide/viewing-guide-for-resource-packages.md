# Viewing Guide for Resource Packages

## Viewing Community Message Volume Resource Packages

For the community downstream message volume covered by the current [Chat Monthly Package](https://www.tencentcloud.com/document/product/1047/67650), go to the [Chat Console](https://console.trtc.io/chat), select **Chat > Configuration >** [**Group Configuration**](https://console.trtc.io/chat/qun-setting) in the left sidebar, **select the target application at the top**, and **view it under Group feature configuration > Community**.**.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/bf69872bcb5711f0a4a55254001c06ec.png)

## Viewing Plug-In Resource Packages

For usage viewing of plug-ins such as cloud search and speech-to-text after activation, go to the [Chat Console](https://console.trtc.io/chat/plugin), select **Chat >**[**Advanced Features**](https://console.trtc.io/chat/plugin) in the left sidebar, **select the target application at the top**, click on any plugin card, and enter the plugin details to view **.**
