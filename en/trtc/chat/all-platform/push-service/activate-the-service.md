# Activate the Service

## Activating the Trial Service

In order for you to better experience the features of the Push service, we offer a 7-day trial version for each SDKAppID for free. Each SDKAppID can enjoy the trial once, with each trial valid for 7 days.

You can activate the trial version of the Push service in the Chat Console. The specific steps are as follows:

1. Log in to the [Chat Console](https://console.trtc.io/), and click **Create Application**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/616616dd7a4211ef9e7a5254000e0cea.png)

2. Enter your application name in the **Create Application** dialog box, and click **Create**. In the creation popup, select **Chat** and enter the application name, choose the data storage **Region**, and click **Create**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/417a6aee7a4211ef80ff525400d5f8ef.png)

3. After completing the Chat application creation, go to [Chat > Push > Access Settings](https://console.trtc.io/chat/push-plugin-push-identifier) page, click free trial to get the trial version of the Push service.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/753a37d5c35f11efbcd1525400bf7822.png)

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b954ef0abc3f11efba8d525400f69702.png)

4. After successfully claiming, you can view the remaining valid time of the trial version on the page.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7dcaf3ddc35f11efbaf8525400454e06.png)

## Purchasing the Official Version

The price description of the Push service monthly package is as follows:

> **Note:**The Push service is provided as a plugin for the Chat product. When purchasing the Push service monthly package, you must also purchase the paid version of Chat to use it.

| Billable Item | Billing Mode | Fee Description |
| --- | --- | --- |
| Push Service (Push) | Resource plan fee(Prepaid) | $299 USD/Month(Among them, regular push has no limit on the number of messages, universal/tag push API free calls 100 times/day, universal/tag push free sends 30 million messages/month) |
|  | Out-of-plan resource usage fee(Postpaid) | Universal/Tag push interface call increase, each additional call/day, Post-payment after Daily Settlement: $10 USD/day.Universal/Tag push messages, each additional 10,000 messages, post-payment after monthly settlement: 0.1 USD/10,000 messages. |

**If you need to purchase the Push service (Push) monthly package, please follow these steps:**

1. Go to the [Chat purchase page](https://buy.tencentcloud.com/avc), select the correct data center and the application (SDKAppID) you want to purchase, choose the Chat monthly package (Standard or Pro or Pro-plus or Enterprise) and add the Push Notifications package, then place the order.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/47d9d814bc4f11ef96d352540075b605.png)

2. After the purchase is complete, you can go to the [Push Overview](https://console.trtc.io/chat/push-plugin-push-identifier) to view the application information.
