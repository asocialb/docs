# React Native

## 1.0.0 (2024-12-4)

- Added API `disablePostNotificationInForeground`, toggle notification bar notifications when the app is in the foreground (default is on).
- Added API `createNotificationChannel`, support for OPPO/FCM to set custom notification channels.
- Added notification bar click event `Push.EVENT.NOTIFICATION_CLICKED`.
- Optimized access experience.
- Supports custom ringtones feature.

## 0.3.1 (2024-10-15)

- update in readme android access process

## 0.2.0 (2024-10-11)

- Added API `addPushListener/removePushListener`, supports receiving online push messages and push message recall notifications.

## 0.1.0 (2024-09-10)

- For React Native, based on Tencent Cloud Push Service, supports iOS and Android push, and adapts to major manufacturers' push services
