# Billing Overview

## Billing Components

Chat offers two billing methods:

- Package prepaid: Purchase first, then deduct. Purchase by cycle, valid from day T to day T of the next month. For example, if purchased on March 1, 2023, it is valid from March 1, 2023, to April 1, 2023.
- Pay-as-you-go: Billed by calendar month. Fees incurred in a calendar month will be deducted within the first 3 days of the next month.
