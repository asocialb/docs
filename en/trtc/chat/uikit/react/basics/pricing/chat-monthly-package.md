# Chat Monthly Package

To provide you with better service, Tencent RTC Chat launches new **prepaid monthly packages** from **Beijing Time January 8, 2025** (**Trial, Standard, Pro (formerly Advanced, now renamed Pro), Pro Plus, and Enterprise**). You can unlock different value-added features for different applications (SDKAppID) based on actual business needs. For details, refer to [Features and Billing Instructions](#function).

## Chat Monthly Packages

### Billing Instructions

- The payment method is prepaid. It adopts a purchase-first, deduct-later approach.
- A monthly package is valid from the purchase date to the same day next month. For example, if purchased on January 1, 2025, it will be valid from January 1, 2025, to February 1, 2025.
- Each package is only effective for a single SDKAppID. If you have multiple SDKAppIDs to purchase, you need to [buy](https://console.trtc.io/subscription/buy/chat?packType=pro&language=en) a separate monthly package for each sequentially.
- After the monthly package purchased for a single SDKAppID expires, the unlocked value-added features will also be disabled and downgraded to Trial Version. It is recommended to enable auto-renewal after purchasing the monthly package to avoid affecting business use. To enable automatic renewal: Go to the [purchase page](https://console.trtc.io/subscription/buy/chat?packType=pro&language=en) -> select Deployment Region/Application/Package editions > Enable automatic renewal.

### Billing Example

User A wants to use cloud search, so they need to purchase **Pro Plus**Edition. They bought **Pro Plus**Edition for 1 month on April 1, 2025, and enabled the **automatic renewal service**.

then:

1. The customer makes a one-time payment of 1,299 USD.
2. Starting from April 24, 2025 (renewal 7 days in advance), **auto-renewal** is triggered, and 1,299 USD is automatically deducted each month. **Pro Plus**Edition is issued, and the validity period of the renewed version is from May 1 to June 1.
