# DismissRoomByStrRoomId

## 1. API Description

Domain name for API request: trtc.intl.tencentcloudapi.com.

This API is used to remove all users from a room and close the room. It works on all platforms. For Android, iOS, Windows, and macOS, you need to update the TRTC SDK to version 6.6 or above.

A maximum of 20 requests can be initiated per second for this API.

## 4. Example

### Example1 Closing a room

This example shows you how to close room `abcd`.

#### Input Example

```
https://trtc.intl.tencentcloudapi.com/?Action=DismissRoomByStrRoomId
&SdkAppId=1400000001
&RoomId=abcd
&<Common request parameters>
```
